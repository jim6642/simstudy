## 🎮 游戏简介

**SimStudy（期末冲刺！）** 是一款以大学生期末考试周为主题的 **Roguelike 文字冒险 + 反应小游戏**。

3天倒计时 × 3时段/天 = 9回合。选地点→选行动→10秒emoji点击小游戏→随机事件→结算→商店→重生。

核心是**精力管理**——连续学习消耗精力，精力过低学习效率暴跌；娱乐回复精力但轻微扣学识。

## 🎯 属性系统

| 属性 | 上限 | 起始 | 说明 |
|------|------|------|------|
| 📚 学识 | 100 | 0 | 通过学习获得 |
| ⚡ 精力 | 100 | 60 | <30效率打7折，<15打4折 |
| 🎯 成绩 | 100 | 20 | 学识×0.55 + 精力×0.20 累加 |
